It would be nice to have collapsible nodes and edges.

If the graph to be rendered exceeds some threshold, say 100 nodes, then
all nodes with more than X direct children should be collapsed. A badge
should indicate the number of child nodes, and double clicking should
toggle the collapse state. X should be configurable, but in a first
draft of this feature this can be a variable.

Similar with edges, even though I believe they can always be collapsed
if there is more than one edge in the same direction. The label should
indicate that there is more than one edge. Double clicking should toggle
collapse state.
